# Technologies

This project uses the following technologies:

- **Next.js 16** - React framework for server-side rendering and routing
- **MUI (Material UI) v7** - React component library for UI components
- **Tailwind CSS v4** - Utility-first CSS framework for styling
- **Axios** - HTTP client for making API requests
- **React Query (TanStack Query) v5** - Data fetching and state management
- **Redux Toolkit** - State management for complex application state
- **React Hook Form** - Form management and validation
- **Zod** - Schema validation library
- **TypeScript** - Type-safe JavaScript superset

All new features and components must be built using these technologies.
